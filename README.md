# Mini SQL Translator

A simple web-based project that converts natural language queries into SQL queries.

## 🚀 Features
- Converts user input into SQL queries
- Beginner-friendly interface
- Fast and simple processing

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 📌 Example
Input: Show all students  
Output: SELECT * FROM students;
